## Space Invaders - Java
Space Invaders game in Java

## How to run
1. Navigate to the project directory:
   ```bash
   cd SpaceInvaders
   ```
2. Compile the Java class from the src into the bin folder
   ```bash
   javac -d bin src/*.java
   ```
3. Copy the img folder to bin
   ```bash
   cp -r img/ bin/
   ```
4. Run the App class
   ```bash
   java -cp bin/ App
   ```
